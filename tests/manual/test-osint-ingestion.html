Potential False Positives (Should be correctly handled)

Version numbers like 1.2.3.4 or 192.168 (incomplete) should not be detected as IPs.

Product codes like ABC-123-DEF or order numbers #12345678901234567890 are not crypto addresses.

Time formats like 12:34:56:78 should not be detected as MAC addresses.

Hashtags like #trending or #tech2024 are not social media handles.

Mathematical expressions like f(x) = 2x + 3 @ point x=5 are not email addresses.
